Getting the SDK

If you have an up-to-date version of Dart Editor, you already have the SDK. Look for a dart-sdk directory inside of the Dart Editor directory.

If you don't have Dart Editor or want a different version of the SDK, download the ZIP file for your platform's SDK:

{% include downloads/_sdk.html %}

Or live dangerously: download the {% for sdk in site.custom.downloads.binaries %} {{ sdk.os }} latest, untested build of the {{ sdk.name }} 64 bit SDK or the {{ sdk.name }} 32 bit SDK. {% endfor %} These continuous builds usually are updated daily.

After you download and unpack the SDK, you'll have a directory named dart-sdk. You'll see a bin directory for the command-line tools, and a lib directory for the Dart libraries.

SDK contents

The SDK includes the Dart libraries and the following command-line tools:

  • Dart-to-JavaScript compiler (dart2js)
  • Dart VM (dart)
  • Dart package manager (pub)
